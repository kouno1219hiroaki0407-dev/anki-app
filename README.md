# 🧠 Anki風 暗記カードアプリ（PWA対応）

シンプルで軽量、そしてオフラインでも使える暗記カードアプリです。  
Ankiのように「問題と答え」を登録し、繰り返し学習できます。  
PWA対応のため、スマホやiPadのホーム画面に追加してアプリとして利用できます。

## ✨ 主な機能

- 問題と答えの登録
- カード一覧の表示（ランダムカラー）
- IndexedDB によるローカル保存
- PWA対応（オフライン動作）
- エクスポート／インポート機能（バックアップ・端末移行）

---

## 📤 データのエクスポート（バックアップ）

カードデータを JSON 形式で保存できます。

### 手順
1. 画面右上のメニューを開く  
2. 「エクスポート」を選択  
3. `anki-cards.json` がダウンロードされます  
4. このファイルを保存しておけば、いつでも復元できます

---

## 📥 データのインポート（復元）

以前エクスポートした JSON を読み込むことで、カードを復元できます。

### 手順
1. メニュー → 「インポート」  
2. `anki-cards.json` を選択  
3. 読み込み完了後、カード一覧に反映されます

### 注意点
- インポートは既存データに **追加** されます  
- 重複カードが発生する場合があります  
- JSON 以外のファイルは読み込めません  
- データは端末ごとに独立しています（IndexedDB）

## 📱 PWAとして使う方法（iPhone / iPad）

1. Safariで公開URLを開く  
2. 共有ボタン → 「ホーム画面に追加」  
3. アイコンをタップすると、アプリとして起動します  
4. オフラインでも利用可能です
